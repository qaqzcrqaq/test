1、git clone 仓库名        #从远程仓库把项目拿下来

2、git checkout 分支名     # 切换到某分支

3、git add ./git add *.txt    #增加

4、git commit -m "提交信息"    #提交到本地仓库

5、git push origin 分支名      #提交到远程仓库
